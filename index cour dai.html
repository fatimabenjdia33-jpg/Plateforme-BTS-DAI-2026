⚙️ Structure & Pointeurs en Algorithmique C

🎥 Prof Language C on YouTube
📖 Leçon: Les Tableaux et les Pointeurs
Language C hwa l-sass dyal l-informatique. L-pointeur hwa wa7ed l-variable li kaykhbi l-adresse dyal variable khor f l-mémoire RAM (`&valeur`).
#include <stdio.h> int main() { int age = 19; int *ptr = &age; // ptr fih l-adresse dyal age printf("Valeur de age: %d\n", *ptr); return 0; }

🗄️ Gestion des Bases de Données (Requêtes SQL complexes)

🎥 Prof SQL on YouTube
📖 Leçon: Les Jointures (JOIN) & Regroupements (GROUP BY)
Kandiro INNER JOIN bash njm3ou bin 2 tableaux b l-clé primaire o l-clé étrangère. GROUP BY kandiroha m3a count() ou sum() bash n-calculew l-majmou3 standard.
-- Requête pour afficher le nombre d'étudiants par ville SELECT Ville, COUNT(*) AS TotalEtudiants FROM Etudiants GROUP BY Ville HAVING COUNT(*) > 1;
📝 Examen National Blanc (SQL)
Structure MLD:
Filiere(id_filiere, nom_filiere)
Stagiaire(num_inscription, nom, prenom, #id_filiere)
-- Requête: Trouver les noms des stagiaires inscrits en filière 'DAI' SELECT S.nom, S.prenom FROM Stagiaire S INNER JOIN Filiere F ON S.id_filiere = F.id_filiere WHERE F.nom_filiere = 'DAI';

💡 Kifach tnmedi wa9tek o t-majori f l-BTS DAI ?

L-mfield dyal l-informatique fih l-pratique bzzaf, dakchi 3lach had l-khotta hiya l-a7san :

  • 1. Qanoun 70/30 (70% Pratique, 30% Théorie): Bla matb9ay ghir tqray l-leçons, 7lli VS Code o bday t-taper l-code o t-troubleshooti l-machakil rassek.
  • 2. Technique Pomodoro: Khdmi 25 dqiqa focus (bla tlfn) o rta7i 5 dqayq. Katkhlli l-39l y39l 3la l-algorithmes dghya.
  • 3. T9sim l-Mawad f l-Ousbou3:
    • L-Itnayn o l-Trata2: Khassihom l l-Logiciels o Coding (Python / Language C).
    • L-Arbi3a2: Khassih l l-Bases de données (SQL) o l-Modeling.
    • L-Khmis o l-Joumo3a: Khassihom l Frontend o PHP (Multimedia).
    • L-Sbt: Révision o T9ad dyal l-projets li khedama 3lihom.
  • 4. Projets Réels: B7al had l-platforme li kaddiri daba, hiya li ghadi t-bni lik CV s7i7! 🔥
